# Mô hình ngôn ngữ lớn (Large Language Models, LLM)

> [Knowledge Tree](../../README.md) / [Trí tuệ nhân tạo (AI)](../README.md) / **Mô hình ngôn ngữ lớn (LLM)**

## Vai trò trong cây kiến thức

*Mô hình ngôn ngữ lớn* (*Large Language Model — LLM*) là mô hình AI được huấn luyện trên lượng lớn dữ liệu ngôn ngữ để hiểu, tóm tắt, sinh và biến đổi văn bản. Các kỹ thuật như **RAG**, **Chain of Thought (CoT)**, *prompt engineering*, *tool calling*, *fine-tuning* và **Minimal Reproducible Example (MRE)** giúp đưa LLM vào ứng dụng thực tế một cách đáng tin cậy, có thể kiểm thử và dễ điều tra lỗi hơn.
